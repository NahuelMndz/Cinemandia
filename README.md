# Cinemandia - es una aplicación web creada con React que consume una API de películas para mostrar información actualizada sobre films y series.

Este proyecto demuestra mi capacidad para integrar APIs y el uso de tailwindcss

# Tecnologías utilizadas

Frontend: JavaScript, React, tailwindcss

API : themoviedb API

Diseño: Primera ves usando tailwindcss y creando un diseño responsive con tailwindcss.
